Comment obtenir le fichier d'export ?

  1. Installez l'extension SW Exporter sur Chrome ou Firefox
  2. Connectez-vous à Summoners War sur votre navigateur
  3. Cliquez sur l'icône SW Exporter et exportez vos données
  4. Glissez le fichier .json dans la zone ci-dessus

⚠️ Les données sont stockées uniquement dans votre navigateur (localStorage). Elles ne sont jamais envoyées à un serveur.
